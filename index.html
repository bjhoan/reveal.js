Archivos Planos

¿Que son los Archivos Planos?

Los archivos planos se pueden describir como bases de datos de texto plano. Cada resgistro de un archivo plano esta en una linea diferente, lo que facilita la maniobra a traves de los datos de ellos.

caracteristicas de los archivos planos

  • Son faciles de configurar y de integrar.
  • Los archivos planos almacenan todos los registros en un solo lugar de una manera facil de entender.
  • No ocupa muchos recursos, perfecto para empresas pequeñas.

XML

Extensible Markup Language(XML) es un lenguaje de marcado que define un conjunto de reglas para la modificacion de documentos.

caracteristicas de XML

  • XML almacena los datos en formato de texto simple, lo que facilita el almacenar datos.
  • Simplifica el intercambio de datos entre aplicaciones compatibles
  • Permite el acceso de diversas aplicaciones a tus datos, no solo HTML

XML permitio la creacion de otros lenguajes

Muchos idiomas nuevos son creados con XML, aqui muestro algunos:

  • XHTML(Extensible HyperText Markup Language)
  • WSDL (Web Services Description Language)
  • RSS (Really Simple Syndication)

Json

JavaScript object notation (Json) es un formato de intercambio de datos que resulta muy facil de leer y escribir para programadores y sencillo de interpretar y crear para las maquinas.

Json

json es un lenguaje autodescriptivo y facil de entender, siendo una altenativa para XML, siendo mas rapido en cualquier computador, otro factor por el que tambien se destaca por ser mas ligero en las transmisiones.

json

A pesar de todo Json tambien tiene sus desventajas, una de estas es que no soporta grandes cargas de datos, requiere de mecanismos externos para la seguridad y no cuenta con extensibilidad.

Parquet

Apache Parquet es un formato de almacenamiento en columnas enfocado en optimizar el procesamiento y el modelo de los datos.

caracteristicas de Parquet

  • Los archivos parquet se diseñaron teniendo estructuras complejas de datos
  • Esta licenciado bajo la licencia apache y disponible para cualquier proyecto
  • Esta orientada a columnas y diseñado para brindar un almacenamiento en columnas.
